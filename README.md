# Plant Disease Detection using Deep Learning (91% Accuracy)

## 🌿 Project Overview
This project is a Data Science solution for automated plant pathology diagnosis. Using a **SqueezeNet 1.1** Convolutional Neural Network (CNN), the system classifies **29 different categories** of plant diseases across various species like Tomato, Bell Pepper, Cherry, and Apple.

## 📊 Key Results
- **Accuracy:** 91.45%
- **Model:** SqueezeNet 1.1 (Lightweight & Efficient)
- **Categories:** 29 unique Plant-Disease pairs.
- **Optimizer:** Adam

## 🚀 Technical Highlights
- **Data Engineering:** Custom deep-directory scanning for hierarchical datasets.
- **Preprocessing:** Z-score normalization and image resizing to 128x128.
- **Inference:** A diagnostic tool that separates plant species from disease names for better readability.

## 📁 Repository Structure
- `train_deep_learning.py`: Script for model training and hyperparameter tuning.
- `demo.py`: User-friendly inference script for real-time diagnosis.
- `Final_Report.pdf`: Detailed 10-page Data Science documentation.
